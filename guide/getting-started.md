# Getting Started

Welcome to the documentation! This guide will help you get started with using this documentation site.

## Installation

First, install the dependencies:

```bash
npm install
```

## Development

Start the development server:

```bash
npm run docs:dev
```

The site will be available at `http://localhost:5173`

## Build

Build the site for production:

```bash
npm run docs:build
```

## Preview

Preview the production build locally:

```bash
npm run docs:preview
```
